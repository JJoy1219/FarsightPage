About Farsight

Farsight is a powerful machine learning model designed to provide deep insights and analytics.

Key Features

  • Real time game state analysis
  • Makes predictions on potential winrate based on suggested in-game actions
  • Trained on thousands of games to predict win chance based on gamestates
